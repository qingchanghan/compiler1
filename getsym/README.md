## 词法分析

### 范围限制

```c
#define ILNGMAX 30 //标识符长度最大值
#define STRLNGMAX 100 //字符串长度最大值
#define MAX_DIGIT 9 //int最大位数
```

### 错误处理

| 编号   | 内容          |
| ---- | ----------- |
| 0    | 标识符超长       |
| 1    | 字符错误        |
| 2    | 单引号内只能有一个字符 |
| 3    | 字符串超长       |
| 4    | 缺少双引号"      |
| 5    | 数字过大        |
|      |             |
